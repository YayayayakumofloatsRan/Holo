## What Stage78 Adds

Stage78 turns the Stage70-77 biomimetic evidence chain into an explicit,
falsifiable neuroscience correspondence matrix. It does not add a new runtime
mechanism and does not spend more provider tokens by default. The new evaluator
reads the Stage77 model-family stability report and maps current Holo variables
to four theory families:

- global neuronal workspace ignition and broadcast
- hippocampal indexing and complementary learning systems replay
- predictive-processing precision-weighted correction
- neuromodulatory adaptive gain

The implementation is `holo_host/biomimetic_theory_correspondence.py` with CLI:

