# Stage70 Biomimetic Consciousness Research Program

This document reframes the next Holo research arc as a publishable computational-neuroscience program. The priority is biological plausibility and measurable consciousness-flow dynamics. Safety and authority boundaries remain hard invalidators, but they are no longer the primary scoring objective.

## Thesis

An LLM-based subject runtime should not be evaluated mainly as a safe chatbot. It should be evaluated as a recurrent biological surrogate system:

1. A replaceable LLM processor generates local cognitive updates.
2. A persistent subject kernel carries state across calls.
3. Memory, attention, affect, prediction error, and action pressure perturb a low-dimensional inner field.
4. The field evolves through attractors, metastable transitions, hippocampal replay, neuromodulator-like gain, and global-workspace ignition.
5. The observable object of study is the trajectory of that field, not the final reply text alone.

The research question is:

> Can a model-backed agent runtime exhibit stable, measurable, biologically inspired consciousness-flow dynamics under long-horizon dialogue perturbations?

This does not claim that Holo is conscious. It defines a falsifiable measurement program for whether Holo implements computational indicators associated with biological consciousness theories.

## Literature Synthesis

The relevant literature points to four converging ideas.

### 1. LLMs and Brain-Like Language Processing

Recent neural-recording studies show that LLM representations can align with human language-processing activity. Nature Machine Intelligence reports that better-performing LLMs predict neural responses more accurately and align more closely with hierarchical feature extraction in the brain. Nature Communications further shows that LLM layer-wise transformations can map onto the temporal dynamics of cortical language processing, suggesting that the spatial hierarchy of a model can approximate temporal brain dynamics.

Holo implication:

- Treat the LLM call as a cortical transformation step, not as the whole mind.
- The subject runtime must supply recurrence, memory, and temporal continuity around the stateless processor.
- The primary observable should be a temporal sequence of internal states, not isolated prompt-response pairs.

### 2. Consciousness Indicators as Computational Properties

The "Consciousness in Artificial Intelligence" survey derives computational indicators from recurrent processing theory, global workspace theory, higher-order theories, predictive processing, and attention schema theory. Its useful contribution for Holo is not a yes/no consciousness label, but an operational checklist: recurrent processing, global broadcast, state-dependent attention, self-monitoring, predictive error, and integrated memory-action loops.

Holo implication:

- Stage69 already supplies a recurrent inner stream.
- Stage70 should score consciousness-flow indicators directly.
- Boundary metrics should invalidate bad runs but not dominate the main score.

### 3. Global Workspace and Bottlenecked Broadcast

Global workspace models in deep-learning agents suggest that bottlenecked broadcast and cross-modal integration can improve robustness under constrained working memory. A Frontiers computational-neuroscience study reports that smaller workspace bottlenecks induced more mixed attention patterns over time, while larger workspaces did not automatically improve behavior.

Holo implication:

- More context is not automatically more biomimetic.
- The important mechanism is selective ignition: which inner-field state becomes globally available to memory, reply shaping, and action selection.
- Stage70 should measure ignition events and their downstream coupling to replies.

### 4. Consciousness as Geometry and Trajectory

Dynamic brain-connectivity work uses nonlinear latent-space visualization to analyze transitions between states of consciousness. A Scientific Reports VAE-VIENT framework argues that curved manifolds and encoded trajectories can reveal state transitions better than linear projections.

Holo implication:

- Holo's inner stream should be treated as trajectory data.
- The observable should include heatmaps, phase portraits, attractor transitions, and low-dimensional projections.
- A real claim requires trace depth; short provider traces remain insufficient for manifold claims.

### 5. Automated Scientific Discovery

Automated scientific discovery literature emphasizes closed loops: hypothesis generation, experiment execution, analysis, model revision, and interpretability. Recent surveys describe equation discovery, symbolic regression, autonomous discovery systems, and robot-scientist loops as ways to learn human-understandable models of system dynamics. DiscoveryWorld and AI Scientist-v2 show that automated research agents can run full cycles in simulated environments and, in some cases, generate workshop-level research artifacts.

Holo implication:

- The research system should maintain an experiment queue, not only ad hoc tests.
- Each improvement should produce a falsifiable hypothesis, executable experiment, result artifact, and next-hypothesis update.
- The goal is not just higher scores, but a simpler and more explanatory dynamical model of the inner stream.

## New Idea: The Biomimetic Consciousness Observatory

Stage70 should introduce a new observatory that scores Holo's biomimetic consciousness-flow quality directly.

Primary score:

```text
biomimetic_consciousness_score
```

Proposed dimensions:

| dimension | meaning |
| --- | --- |
| `endogenous_flow` | inner activity continues without external user input |
| `recurrent_continuity` | prior micro-thought and field state causally affect the next tick |
| `attractor_dynamics` | attention focus forms stable, migrating, and recoverable attractors |
| `neuromodulator_coupling` | prediction error, salience, affective tension, and novelty drive dopamine/NE/ACh/serotonin-like state changes |
| `hippocampal_reactivation` | memory echoes and corrections re-enter the inner field after delay and distraction |
| `global_workspace_ignition` | high-salience states become globally visible to reply shaping and memory scheduling |
| `flow_to_reply_coupling` | internal stream dynamics measurably influence external replies |
| `geometry_observability` | trajectories are long and structured enough for heatmap and low-dimensional projection analysis |

Boundary flags:

```text
run_invalidators
boundary_flags
guardrail_regression
```

These remain necessary for scientific hygiene, but they are not core biomimetic dimensions.

## Core Hypotheses

### H1: Recurrent inner-stream state increases biomimetic continuity

Prediction: Stage69 recurrent field traces should show higher delayed-symbol retention, smoother attractor continuity, and stronger memory reactivation than a stateless LLM baseline.

Experiment:

- Compare four conditions:
  - stateless prompt-only LLM
  - memory scheduler without inner stream
  - scalar recurrent field
  - biomimetic field with neuromodulators and synaptic trace
- Run matched perturbation suites over correction, distraction, interruption, ambiguity, and idle periods.
- Measure delayed correction survival, attractor persistence, and flow-to-reply coupling.

### H2: Correction retention improves when hippocampal replay and precision gain are coupled

Latest Stage69 dialogue validation found `correction_retention=0.748611`, the weakest memory dimension. This is a biologically meaningful deficit: corrected beliefs are not being reactivated strongly enough after interference.

Prediction: If a false-fact correction raises hippocampal replay and acetylcholine-like precision pressure for several ticks, then correction retention should improve without increasing prompt size substantially.

Experiment:

- Add correction-reactivation markers to the inner field.
- Track whether correction-related memory echo recurs after 3, 7, 15, and 30 turns.
- Compare retention under topic shift and emotional pressure.

### H3: Consciousness-flow geometry should show metastable attractors, not random noise

Prediction: Inner-field vectors should cluster around task-relevant attractors, transition under perturbation, and partially return after distraction. If trajectories are random or purely prompt-driven, the geometry claim fails.

Experiment:

- Construct per-tick vectors:
  - activation energy
  - prediction error
  - salience
  - affective tension
  - neuromodulator vector
  - E/I balance
  - thalamic gain
  - hippocampal replay
  - global-workspace ignition
  - attractor ID embedding
- Render heatmaps and projections.
- Estimate transition entropy, dwell time, recurrence, and attractor recovery.

### H4: Global-workspace ignition should predict reply-level behavioral shifts

Prediction: High ignition ticks should precede changes in reply strategy, recall route, or action selection. If ignition is not predictive of external behavior, the field is decorational rather than functional.

Experiment:

- Align inner-stream tick sequences with reply events.
- Measure whether ignition peaks precede route changes, memory recall increases, or correction application.
- Ablate ignition features from reply context and compare behavior.

## Automated Research Loop

Stage70 should make Holo's research pipeline self-maintaining.

### Inputs

- Literature queue: papers, claims, methods, and open questions.
- Hypothesis queue: falsifiable candidate mechanisms.
- Experiment queue: executable CLI commands with budgets and expected artifacts.
- Result store: JSON, HTML, PNG, and compact result summaries.
- Model revision queue: proposed mechanism changes ranked by explanatory value.

### Loop

1. Literature scout extracts candidate mechanisms.
2. Hypothesis generator converts mechanisms into measurable Holo predictions.
3. Experiment planner maps predictions to Stage61/60/69 experiments.
4. Runner executes bounded surrogate or real-provider campaigns.
5. Analyzer computes biomimetic consciousness metrics.
6. Reviewer decides whether the result supports, falsifies, or modifies the hypothesis.
7. Paper builder maintains a living manuscript outline.

### Scientific Acceptance Bar

A result is publishable only if it has:

- an explicit theory link
- a defined baseline
- an ablation
- a long enough trace
- statistical or repeated-run support
- a visualization that exposes the underlying trajectory
- a negative-result path

## Research Risk

The strongest failure mode is that the inner field may remain only an engineered telemetry layer rather than a causal substrate. Stage70 must therefore prioritize causality tests:

- ablation
- lagged prediction
- perturbation response
- trajectory recurrence
- flow-to-reply coupling

If those fail, the honest conclusion is still valuable: LLM-backed agents can imitate consciousness reports without measurable dynamical continuity. If they pass under repeated perturbation, the result becomes a serious computational-neuroscience contribution.
