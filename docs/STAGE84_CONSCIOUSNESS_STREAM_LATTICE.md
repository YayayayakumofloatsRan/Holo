# Stage84 Consciousness Stream Lattice

## What Stage84 Adds

Stage84 implements the first read-only consciousness-stream latent-dynamics
observatory over the completed Stage83 publication bundle and Stage59/60-gated
real-provider traces. It moves the research object from isolated control
outcomes to continuous stream-state dynamics.

The implementation is `holo_host/consciousness_stream_lattice.py` with CLI:

```powershell
python -m holo_host --config .holo_host.toml evaluate-consciousness-stream-lattice --publication-json artifacts\stage83\stage83_biomimetic_publication_bundle.json --trace-json artifacts\stage77\stage77_ignition_reply_20260515\cells\01_deepseek-v4-pro\provider_trace.json --trace-json artifacts\stage77\stage77_ignition_reply_20260515\cells\02_deepseek-v4-flash\provider_trace.json --output artifacts\stage84\stage84_consciousness_stream_lattice.html
```

## Stream Metrics

Stage84 derives one compact `stream_state` per turn with:

- phase
- salience
- normalized recall budget
- consolidation priority
- prediction-error proxy
- global-workspace ignition proxy
- reply-coupling proxy
- action-score proxy
- internal versus external orientation axis

It then computes:

- `mean_dwell_time`
- `transition_entropy`
- `mean_event_boundary_score`
- `reactivation_return_rate`
- `ignition_report_transfer`
- `active_inference_delta`

## Interpretation

Stage84 supports this bounded claim:

```text
Holo's real-provider Stage77 traces can be represented as a low-dimensional
latent stream-state lattice with measurable dwell, transition, event-boundary,
marker-dependent reactivation-return, and action-clamp stream-movement
proxies.
```

The GNW claim remains partial:

```text
Stage84 does not yet support stable ignition-to-report transfer in the real
Stage77 traces. The current real-trace `ignition_report_transfer` is `0.0`, so
GNW must remain a partial access/report proxy until explicit stream-to-report
instrumentation is added and replicated.
```

The result remains bounded:

```text
Stage84 is not evidence of subjective consciousness, biological brain dynamics,
provider-native neural dynamics, or an implemented biological global workspace.
```
