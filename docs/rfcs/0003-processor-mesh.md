# RFC 0003: Processor Mesh

## Status
Accepted

## Decision
`codex exec` is promoted from a reply-only helper to a task-typed processor mesh interface.

## Initial Tasks
- `reply`
- `recall_reconstruct`
- `memory_consolidate`
- `reflect`
- `dream`
- `initiative_plan`
- `self_check`

## Rules
- the program owns stable state and write policy
- the processor handles high-level language reasoning
- each task must declare allowed data layers, output shape, and whether writeback is allowed
