# GitHub Bootstrap

This machine does not currently have a Git remote wired to this repo, and `gh` is not installed. The local GitHub scaffolding is ready, but the private repo still needs one manual bootstrap.

## Manual Bootstrap
1. Create a new private GitHub repository.
2. Add it as `origin`.
3. Push the current branch.
4. Create milestones:
   - `M0 Repo + Docs Bootstrap`
   - `M1 Memory Substrate`
   - `M2 Retrieval + Mind Packet V3`
   - `M3 Processor Mesh`
   - `M4 Consciousness Streams`
   - `M5 Social Memory + Autonomy`
   - `M6 Cutover + Decommission`
5. Create labels:
   - `epic`
   - `mind-os`
   - `memory-graph`
   - `retrieval`
   - `processor-mesh`
   - `stream`
   - `wechat`
   - `migration`
   - `docs`
   - `release-blocker`

## After Bootstrap
- use the issue templates under `.github/ISSUE_TEMPLATE/`
- use `.github/pull_request_template.md`
- put prerelease notes under `docs/releases/`
