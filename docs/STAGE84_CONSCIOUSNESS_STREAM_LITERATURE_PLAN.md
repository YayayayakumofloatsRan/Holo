# Stage84 Consciousness-Stream Literature Plan

## Research Pivot

Stage84 should shift Holo from isolated biomimetic controls toward a
stream-of-consciousness latent-dynamics program. The working hypothesis is:

```text
Holo should be studied as a bounded artificial consciousness-flow testbed:
language generation is the report surface, while the subject runtime maintains a
continuous latent stream whose dwell states, event boundaries, reactivation
paths, precision shifts, and ignition-to-report transitions can be measured and
intervened on.
```

This remains an operational proxy program. It must not claim subjective
consciousness.

## Stage84 Mechanism Target

Add a read-only `consciousness_stream_lattice` or equivalent observatory over
existing Stage59/60/77 traces before changing the live runtime:

- `stream_state`: compact vector or feature packet per turn/tick
- `dwell_time`: duration inside a stable latent state
- `transition_entropy`: rigidity versus fragmentation of state transitions
- `event_boundary_score`: surprise/perplexity-style boundary proxy
- `orientation_axis`: internal memory/reactivation versus external task/report
- `reactivation_return_rate`: probability that correction markers restore a
  prior stream state
- `ignition_report_transfer`: probability that an internal stream state reaches
  reply text
- `active_inference_delta`: active-loop stream shift minus passive-input stream
  shift

## Falsification Controls

Stage84 should require at least these controls:

- shuffle stream-state order while preserving prompt cost
- remove correction markers and test reactivation return rate
- neutralize salience and test boundary sensitivity
- clamp gain and test transition entropy
- compare active-inference versus passive replay with matched provider budget

## Acceptance Bar

Stage84 is acceptable only if:

- Stage83 publication bundle remains valid
- stream metrics are reproducible on existing real-provider cells
- at least one direct stream-state control narrows a theory mapping
- GNW remains partial unless ignition-report transfer stabilizes across cells
- all biological and consciousness language stays bounded to operational proxies
- no watcher, live WeChat transport, self-memory, policy, or runtime-decision
  authority is added

## Publication Direction

The publishable framing should be:

```text
A bounded LLM-agent testbed for artificial consciousness-flow proxies, combining
hippocampal-style indexing, predictive precision, neuromodulatory gain, and
partial GNW ignition-to-report controls over real-provider traces.
```

The not-yet-claimable framing is:

```text
Holo is conscious, has subjective experience, or instantiates biological neural
consciousness.
```
